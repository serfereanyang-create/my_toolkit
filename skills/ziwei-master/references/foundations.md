# Zi Wei Dou Shu Foundations

## Open-Source Starting Points

- `SylarLong/iztro`: TypeScript/JavaScript Zi Wei Dou Shu astrolabe/chart library. Treat it as the preferred programmatic source for chart placement when available.
- `x-haose/py-iztro`: Python version of `iztro` for Python workflows when current and installable.
- `ruijayfeng/ziwei`: ZiweiKnows, an open-source Zi Wei Dou Shu charting and AI reading application; useful for product patterns, not as a sole interpretive authority.
- `SylarLong/react-iztro`: React component built on `iztro`; useful when a visual chart component is needed.

When accuracy matters, browse the current repositories or official docs before using APIs because package names and method signatures can change.

## Required Birth Inputs

- Sex/gender as required by the calculator.
- Birth year, month, day, and hour.
- Calendar type: solar/Gregorian or lunar/Chinese.
- Leap lunar month flag if lunar input is used.
- Birthplace/time zone if the date/time conversion might cross a boundary.

Do not silently convert lunar/solar dates. State the convention used.

## Core Chart Objects

- 十二宫: 命宫, 兄弟, 夫妻, 子女, 财帛, 疾厄, 迁移, 交友/仆役, 官禄, 田宅, 福德, 父母.
- 命宫: personality, base pattern, self-positioning.
- 身宫: how the person enacts life path; read together with 命宫.
- 三方四正: the target palace plus its career/wealth/migration-style support palaces depending on palace geometry. Avoid reading any palace in isolation.
- 大限: decade-level trend.
- 流年: year-level activation.
- 四化: 化禄, 化权, 化科, 化忌. Always identify whether they are natal, decade, annual, or flying transformations.

## Major Stars

Fourteen major stars commonly used:

- 紫微: authority, organization, centrality, leadership image.
- 天机: planning, movement, learning, strategy, change.
- 太阳: visibility, giving, public activity, male/yang themes.
- 武曲: finance, execution, discipline, material management.
- 天同: comfort, affinity, softness, enjoyment.
- 廉贞: rules, desire, charisma, boundaries, complexity.
- 天府: storage, resources, steadiness, administration.
- 太阴: assets, emotion, care, female/yin themes, refinement.
- 贪狼: desire, sociability, arts, appetite, versatility.
- 巨门: speech, debate, doubt, hidden friction.
- 天相: service, coordination, image, support roles.
- 天梁: protection, principles, elders, rescue, scrutiny.
- 七杀: pressure, independence, decisive change.
- 破军: disruption, rebuilding, risk, breakthrough.

Interpret a star through its palace, brightness if known, combinations, four transformations, and malefic/benefic support.

## Common Auxiliary Signals

- 左辅/右弼: help, allies, support.
- 文昌/文曲: learning, writing, aesthetics, documents.
- 天魁/天钺: noble support, recognition.
- 禄存: stored resources, conservative wealth.
- 天马: movement, travel, mobility.
- 擎羊/陀罗: pressure, sharpness, obstruction, injury-prone symbolism.
- 火星/铃星: volatility, speed, temper, sudden events.
- 地空/地劫: loss, emptiness, unconventionality, sudden gaps.

Do not overstate a single auxiliary star. Combine it with palace theme and major-star structure.

## Reading Guardrails

- Treat Zi Wei Dou Shu as a cultural/traditional interpretation system.
- Keep predictions probabilistic and contextual.
- Never give deterministic harm claims, medical diagnoses, investment instructions, or legal decisions.
- Prefer "this chart structure often shows..." and "verify against actual experience".
- Ask for the original chart image or exact birth data when the provided data is incomplete.

## Practical Analysis Template

1. Confirm data and charting convention.
2. List chart facts: 命宫/身宫, major stars, four transformations, key palace clusters.
3. Give the main pattern in one paragraph.
4. Analyze the requested topic via the relevant palace and 三方四正.
5. Add timing only if 大限/流年 data is available.
6. End with grounded suggestions and one or two validation questions.
