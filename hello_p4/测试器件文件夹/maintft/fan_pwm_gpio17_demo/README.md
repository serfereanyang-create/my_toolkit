# ESP32-S3 风扇 PWM + SSD1309 显示示例

这个工程适合 Arduino IDE 直接烧录。

功能：
- `ESP32-S3 GPIO17` 输出 PWM，控制 `12V 风扇 + PWM/NMOS 调速模块`
- `SSD1309 128x64 SPI OLED` 实时显示当前 PWM level

## 屏幕参数

- 驱动芯片：`SSD1309`
- 分辨率：`128 x 64`
- 接口类型：`4线 SPI`
- 显示类型：`单色 OLED`
- 供电：`3.3V`

## 当前接线

### 风扇调速模块

- `ESP32-S3 GPIO17` -> 调速模块 `PWM/IN`
- `ESP32-S3 GND` -> 调速模块 `GND`
- `12V+` -> 风扇正极
- 风扇负极 -> 调速模块输出
- `12V 电源负极` -> 调速模块 `GND`
- `12V 电源负极` 要和 `ESP32-S3 GND` 共地

### OLED 屏幕

- `CS2 = GPIO4`
- `FSO = GPIO5`
- `CS1 = GPIO6`
- `DC = GPIO7`
- `SDA = GPIO8`
- `SCL = GPIO9`
- `RES = GPIO10`
- `VCC = 3V3`
- `GND = GND`
- `5VIN = 5VIN`

说明：
- 代码里实际用于显示的是 `CS1/DC/SDA/SCL/RES`
- `FSO` 作为 SPI 的 `MISO` 保留
- `CS2` 当前没有参与显示逻辑

## 当前代码参数

- PWM 输出脚：`GPIO17`
- PWM 频率：`25kHz`
- 分辨率：`8bit`
- 上电策略：先 `100%` 全速启动 `1秒`，再降到目标转速

## 屏幕显示内容

屏幕会显示：
- 当前 PWM 百分比，例如 `Level: 40%`
- 当前占空比，例如 `Duty: 102/255`
- 当前输出脚和频率，例如 `Pin17 25000Hz`
- 一个随 PWM 百分比变化的横向进度条

## 依赖库

Arduino IDE 里需要安装：
- `U8g2`

## 注意事项

- 不要把 `12V` 直接接到 ESP32-S3 引脚
- 风扇电源和 ESP32-S3 必须共地
- 如果风扇低速启动不了，这是常见现象，所以代码里加了先全速启动
- 如果风扇啸叫，可以试试把频率改成 `20kHz` 或 `30kHz`
- 如果屏幕不亮，优先检查 `CS1/DC/SDA/SCL/RES` 和 `3V3/GND`

## 循环效果

- 从 `30%` 逐步升到 `100%`
- 再从 `100%` 逐步降到 `30%`
- 每档停留 `5秒`
